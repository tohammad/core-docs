# Compiler Warning (level 1) CS3003
Type of 'variable' is not CLS-compliant  
  
 A [public](../../../csharp\language-reference\keywords/public.md), [protected](../../../csharp\language-reference\keywords/protected.md), or `protected internal` variable must be of a type that is compliant with the Common Language Specification (CLS). For more information on CLS Compliance, see [Language Independence and Language-Independent Components](../Topic/Language%20Independence%20and%20Language-Independent%20Components.md).  
  
## Example  
 The following example generates CS3003:  
  
```  
// CS3003.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort a1;   // CS3003, public variable  
    public static void Main()  
    {  
    }  
}  
```
